# gpx_to_matplotlib_anim
A Python script to process a GPX file to a two-panel matplotlib animation (panel 1: lat, lon; panel 2: elevation)

## Requirements

Python libraries:
- matplotlib
- numpy

Other tools:
- ffmpeg

## Example Output
